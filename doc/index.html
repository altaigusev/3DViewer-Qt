3DViewer documentation

This is the 3DViewer documentation. Project witten by Jmadie (Alexander Potyanikhin), Wsulu (Nikita Romanoff) and Sshunpik (Svytoslav Gusev).

This version implemented using C++17, Qt 6.2.3

If you want to learn more about technical features CLICK HERE

Main features:

  • Supported 3D format - *.obj
  • Rotation, transition, scaling model by mouse and sliders both
  • Changing colour configuration with form of lines (dashed, solid) and vertices form (circles, squares, none vertices)
  • You can make screenshot of your model in JPG
  • Saving gif screencast (with user actions and automated rotation around Y axis)
  • Texturizing: you can use .bmp textures
  • Global lightspot: you can change position of light, color and intensivity of light with RGB components changing

ILLUSTRATIONS OF WORKING APPLICATION

How to use:

To open .obj file use Folder button

After choosing you will see the message

Wait until model is loaded from disk to RAM. It is important for big models (more than 1M vertices)

The model is opened!

Now you can use right toolbar to change view of the model (move it, rotate, change background color etc.)
